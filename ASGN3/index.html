The bunny lives in the center of the world.

Notes for Grader

Procedural Generation: A unique map layout is generated randomly every time the application starts, creating a different world on each refresh.

Game Objective: The goal is to build a protective shelter for the bunny before night falls.

Time Cycle: The environment transitions based on the number of blocks you place:

  • Day: 0-9 blocks placed.
  • Sunset: 10-14 blocks placed (Hurry up!).
  • Night: 15+ blocks placed.
